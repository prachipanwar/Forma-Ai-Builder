# ✨ Forma AI Builder

An AI-powered low-code form builder built with React, Vite, Tailwind CSS, and Gemini AI.

Forma AI Builder allows users to generate dynamic forms using natural language prompts, customize fields visually, reorder components with drag-and-drop, and preview forms instantly with a modern SaaS-style interface.

## ✨ Features

- 🤖 AI-powered form generation using Gemini AI
- 🧩 Dynamic schema-driven form rendering
- 🎯 Drag-and-drop field reordering
- ⚡ Live preview mode
- 🛠️ Visual field editor
- 📅 Modern date picker with popover calendar
- ⌨️ Keyboard field navigation
- 💾 Local storage persistence
- 📱 Fully responsive builder layout
- 🎨 Modern glassmorphism UI
- 🔔 Toast notifications and loading skeletons

---

## 🧠 AI Generation

Users can generate complete forms using natural language prompts like:

> "Create an employee onboarding form with emergency contact fields"

The application converts prompts into dynamic form schemas using Gemini AI and renders them instantly.
